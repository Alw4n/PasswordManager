# PasswordManager 1.0

Коротко: выпущен релиз **MyPasswordManager 1.0**. Ниже — доступные форматы, какие сборки использовать для Windows и Linux, как проверить целостность и краткие инструкции по установке.

---

## Форматы и рекомендации

| **Формат** | **Windows** | **Linux** | **Примечание** |
|---|---:|---:|---|
| **Uber JAR / fat JAR** | Используйте сборку, собранную на Windows | Используйте сборку, собранную на Linux | Нативные библиотеки (Skiko) зависят от ОС; JAR, собранный на одной ОС, может не содержать нативов для другой |
---

## Установка и запуск

**Windows**

- Скачайте Windows‑сборку (Windows Uber JAR).  
- Для JAR: убедитесь, что установлена **Java 17+**, затем выполните:
```powershell
java -jar PasswordManager_Windows.jar
```
**Linux**

- Скачайте Linux‑сборку (uber JAR).  
- Для JAR: убедитесь, что установлена **Java 17+**, затем выполните:
```Bash
java -version
```
```Для JAR:
java -jar PasswordManager-linux-x86_64.jar
```

##Проверка целостности и безопасность

Мы публикуем SHA256 для каждого артефакта. Проверьте контрольную сумму перед установкой:
```Bash
sha256sum PasswordManager-version.jar
```
Скачивайте релизы только с официальной страницы GitHub Releases.

Что нового в этом релизе

    Основное: стабильный релиз 1.0 — базовый функционал менеджера паролей.

    Исправления: улучшена работа с базой данных, исправлены ошибки UI на Linux.

    Известные ограничения: JAR, собранный на Windows, не будет работать на Linux без пересборки для Linux; аналогично для Windows‑нативов.
